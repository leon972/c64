- C64 WORMS -

Istructions to load and run worms with VICE C64 emulator:

Attach disk image worms.d64 using then VICE menu;
File->Attach disk image->drive 8
then select worms.d64 disk image.

Load the program using this command:

LOAD"WORMS.PRG",8,1

then type

RUN
